What is Python?

Python is a high-level, interpreted, and object-oriented programming language. Unlike low-level programming languages, Python allows developers to write code using simple English-like syntax, making programs easier to read and maintain.


Python follows the philosophy that code should be simple, clean, and easy to understand. This makes it an excellent choice for beginners while remaining powerful enough for professional software engineers.

Installing Python

Getting started with Python is simple. Download the latest version from the official Python website and install it on your computer. During installation, enable the option to add Python to your system PATH. After installation, verify it by opening the command prompt and typing: 

python --version

If Python is installed correctly, the terminal will display the installed version number.


Writing Your First Python Program

The traditional first program is the "Hello, World!" example: 

print("Hello, World!")

The print() function displays output on the screen. It is one of the first functions every Python beginner learns.


Python is an excellent programming language for anyone beginning their coding journey. Its simple syntax, powerful features, and wide range of applications make it one of the best choices for students and professional developers alike. In the next part, we will explore Python variables, data types, operators, user input, and basic programming concepts in detail.

Variables in Python

Variables are used to store data in a program. They allow programmers to save values such as numbers, text, or lists and use them later in the code. Unlike many programming languages, Python does not require you to declare a variable's data type before assigning a value. 


name = "Alice"
age = 20
height = 5.6

print(name)
print(age)
print(height)

In the example above, name, age, and height are variables storing different types of information.


Python Data Types

Every value in Python belongs to a specific data type. Understanding data types helps you choose the correct kind of data for different programming tasks.


  • int – Whole numbers (10, 200, -5)
  • float – Decimal numbers (3.14, 99.5)
  • str – Text values ("Python")
  • bool – True or False values
  • list – Ordered collection of items
  • tuple – Immutable collection
  • set – Unordered unique values
  • dict – Key-value pairs

Taking User Input

Python allows programs to interact with users using the input() function. 

name = input("Enter your name: ")
print("Welcome", name)

The program waits until the user enters information and then stores it in a variable.


Operators in Python

Operators perform mathematical and logical operations on values.

  • Arithmetic Operators (+, -, *, /, %, **)
  • Comparison Operators (==, !=, >, <, >=, <=)
  • Logical Operators (and, or, not)
  • Assignment Operators (=, +=, -=)

a = 10
b = 5

print(a + b)
print(a * b)
print(a > b)

Comments in Python

Comments help explain code and make programs easier to understand. Python ignores comments during execution. 

# This is a single-line comment

print("Hello Python")

Type Conversion

Sometimes you need to convert one data type into another. Python provides built-in functions such as int(), float(), and str() for this purpose. 

age = input("Enter age: ")
age = int(age)

print(age + 5)

Once you understand variables, data types, input, operators, and comments, you are ready to learn Python control structures such as if statements, loops, and functions. These concepts form the foundation of real-world Python programming.

Conditional Statements

Conditional statements allow a program to make decisions based on specific conditions. Python uses the if, elif, and else keywords to execute different blocks of code depending on whether a condition is true or false. 


age = 18

if age >= 18:
    print("You are eligible to vote.")
else:
    print("You are not eligible to vote.")

Conditional statements are used in login systems, calculators, grading systems, games, and many other real-world applications.


Loops in Python

Loops help execute the same block of code multiple times without repeating it manually. Python provides two main loops: the for loop and the while loop. 


for i in range(1, 6):
    print(i)


count = 1

while count <= 5:
    print(count)
    count += 1

Loops are commonly used for processing data, displaying lists, automating repetitive tasks, and solving mathematical problems.


Functions

Functions are reusable blocks of code that perform a specific task. They help organize programs, reduce repetition, and improve readability. 


def greet(name):
    print("Hello", name)

greet("Asim")

Functions can accept parameters and return values using the return statement.
